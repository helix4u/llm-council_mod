# Contributing

Thank you for your interest in contributing to LLM Council!

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/helix4u/llm-council_mod.git`
3. Set up the development environment following the instructions in README.md
4. Create a new branch for your changes: `git checkout -b feature/your-feature-name`

## Development Guidelines

- This project was built as a fun weekend hack and may have rough edges
- Feel free to improve, refactor, or extend as needed
- Code style should be reasonably consistent with existing code
- Test your changes before submitting

## Submitting Changes

1. Make sure your code works
2. Update documentation if needed
3. Commit your changes with clear messages
4. Push to your fork
5. Create a Pull Request

## Notes

This project is provided as-is for inspiration and use. While contributions are welcome, there's no formal maintenance schedule.
